# 视频编辑器实现计划

## 技术选型

- **拖拽排序**：`@dnd-kit/core` + `@dnd-kit/sortable`（轻量 React 拖拽库）
- **数据操作**：复用 `useStoryboardStore` 中已有方法
- **布局**：左侧预览 + 右侧时间轴胶片条

## 整体布局

**移除左侧分集导航侧边栏，全宽给工作区。** 分集切换放在 Header 中的紧凑 Select 下拉框。

```
VideoPage
├── 顶部 Header（shrink-0 border-b）
│   ├── 左：标题"视频合成" + 分集 Select 下拉（紧凑）
│   └── 右：合成配置 | StartCompositionDropdown | 确认进入导出
│
└── 主工作区（flex-1 overflow-hidden，全宽）
    ├── [合成进行中] CompositionProgressPanel（顶部横幅）
    ├── [无镜头数据] VideoEmptyState（居中引导）
    ├── [有已合成视频时] Tabs: 编辑 / 预览
    │   ├── 编辑 Tab → VideoEditor（全宽）
    │   └── 预览 Tab → VideoPlayer
    └── [默认] VideoEditor（全宽编辑器）
└── Dialogs（CompositionConfigDialog + ConfirmCompositionDialog）
```

### 分集选择机制

- 进入页面时从 **URL 查询参数** `?episodeId=xxx` 读取初始分集
- 若无参数则默认选中 **第一集**（`episodes[0]`）
- Header 中的 Select 切换分集时，**同步更新 URL**（`router.replace`）
- 后续计划：分镜页的"确认分镜"按钮可携带 `episodeId` 跳转到 `/project/[id]/video?episodeId=xxx`

## EditorShotItem 设计

每个镜头块包含：

- **缩略图**：`imageUrl` 或灰色占位图 + 镜头序号
- **宽度**：与 `duration` 成比例（`zoom` 控制基准像素/秒）
- **底部**：时长标签 + 字幕文本（截断）
- **悬浮操作栏**：
  - `✂` **切割** — 在中点切割，前半保留属性，后半新建镜头
  - `⊞` **合并下一个** — 时长相加，删除下一镜头（跨场景禁用）
- **选中态**：蓝色边框高亮

---

## 切割 / 合并实现

**切割（Split）**，利用 `storyboard store`：

```typescript
// 1. 将当前镜头时长减半
await updateShot(storyboardId, shotId, { duration: `${halfDuration}s` })
// 2. 在其后插入新镜头（复制属性 + 剩余时长）
await addShot(storyboardId, { ...shot, duration: `${halfDuration}s`, insertAfter: shotId })
```

**合并（Merge with next）**，同一 Storyboard 内相邻镜头：

```typescript
// 1. 更新当前镜头时长 = 当前 + 下一个
await updateShot(storyboardId, shotId, { duration: `${combinedDuration}s` })
// 2. 删除下一镜头
await deleteShot(storyboardId, nextShotId)
```

---

## 拖拽排序实现

- **同场景内**（同一 Storyboard）：`@dnd-kit/sortable` SortableContext → 拖拽结束调用 `reorderShots(storyboardId, newOrder)`
- **跨场景**（不同 Storyboard）：使用 `POST /api/storyboards/shots/move` API，将镜头移入目标场景

跨场景拖拽：`DragOverlay` 跟随鼠标，`DragEnd` 时判断目标容器 id，调用 move API。

## 数据来源

- 镜头数据：`fetchStoryboards(projectId, episodeId)` from `useStoryboardStore`
- 合成任务：`fetchTasks(projectId)` from `useVideoStore`
- 分集列表：`fetchEpisodes(projectId)` from `useVideoStore`

编辑操作（updateShot / deleteShot / addShot / reorderShots）全部通过 `useStoryboardStore` 调用。
